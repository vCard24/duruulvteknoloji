# duruulvteknoloji

Duru ULV Teknoloji Sistemleri — statik kurumsal web sitesi (ürün kataloğu, blog, teklif formu).

## Geliştirme

```bash
node scripts/integrate-seo-content.js   # blog + ürün SEO + sitemap
node scripts/check-internal-links.js    # kırık link kontrolü
```

Canlı sunucuda `api/config.php` dosyasını `api/config.example.php` şablonundan oluşturun (SMTP şifresi repoya eklenmez).
